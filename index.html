The Language of the Web

HTML Overview

HTML or Hyper Text Markup Language is foudnational to almost every web page on the internet. HTML consists of:

*Text Content - the stuff you can see
*Markup - how it appears
*References to other documents - other types of media
*Links to other Pages - Lots of links are strewn across pages like Hello World.

HTML Markup

Things called tags are a set of instructions to make commands to the browser, which starts with an opening tag and closes with a closing tag. Using specific tags, I can make a word bold or make a bold emphasis. These opening and closing tags are called elements.

WhiteSpace

Outside of the tags and attributes, HTML may seem very similar to writing in a word processor. One glaring difference is the use of whitespace, as text on multiple lines will still appear as if they are on the same line unless given specific instruction. Void tags like break can be used to force multiple lines or the paragraph tag can be used, which is wrapped inbetween content and is not a void tag. A break tag is referred to as inline and a paragraph tag is block. Another inline element is span and another block element is div, both service the function of controlling how text is laid out.
